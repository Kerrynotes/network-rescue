# 断网急救 v0.4.3-beta

## 本次修复

- 修复部分 Windows 电脑启动 BAT 时提示“系统找不到指定的路径”或“cho 不是内部或外部命令”。
- 所有用户 BAT 入口改为英文文件名、纯 ASCII 内容和 Windows CRLF 换行。
- 修复托盘网络状态可能一直停留在“正在检查”的问题。
- 启动时先快速识别系统代理、端口和后台 Core，再补充完整联网检测。
- 网络状态新增“本地代理端口归属”和“后台 Core 归属”，便于确认每一层当前属于哪个客户端。
- 扫描超过 45 秒会自动中止并重试，菜单会显示明确错误，不再无期限等待。

## 使用方式

1. 首次安装或升级：双击 `Install-NetworkRescue.bat`。
2. 手动退出后重新启动：双击 `Start-NetworkRescue.bat`。
3. 查看断连历史：双击 `Open-Connection-Logs.bat`。
4. 停止监控并移除开机启动：双击 `Uninstall-NetworkRescue.bat`。

程序界面和说明文档继续使用中文；只有容易受 Windows 编码影响的 BAT 文件改为英文。
