断网急救 v0.4.2-beta——分享版使用说明

一、这个工具解决什么

电脑安装多个代理客户端后，可能出现：

- 客户端界面退出了，核心仍占用 7890；
- 龙猫、全球、唯兔等同源客户端互相影响；
- 系统代理、TUN、DNS 或环境变量分别被不同客户端接管；
- 浏览器、Codex 和终端走不同的旧代理端口；
- 客户端异常退出后无法恢复普通网络。

断网急救只做一件核心事情：

“一键退出全部代理并恢复普通网络”。

成功后，如需代理，请只打开一个客户端，并在客户端里手动选择节点和连接。

二、怎么安装

1. 完整解压压缩包，不要只在压缩包预览里运行；
2. 双击“安装断网急救.bat”；
3. 看到 Windows 托盘区的盾牌图标后即安装完成。

如果你在托盘菜单里点了“退出断网急救”，双击“启动断网急救.bat”即可重新显示盾牌图标；不需要再次安装。

普通安装不需要管理员权限。

只有停止受保护核心、服务或 TUN 时才需要管理员权限。你可以：

- 不安装 Helper：遇到受保护残留时按次确认 UAC；
- 安装 Helper：双击“安装高权限Helper_仅需一次UAC.bat”，只在安装时确认一次 UAC。

三、托盘菜单

1. 网络状态：用“项目 / 当前状态”列表查看当前是谁在代理、端口、TUN 和联网状态；问题与建议会单独编号列出；
2. 一键退出全部代理并恢复普通网络：完整清场；
3. 系统代理异常时自动急救：关闭无人监听的失效代理；其他客户端退出误关主代理时，经端口归属和联网复核后自动重开；
4. 退出断网急救：停止本次后台监控。

断网急救会监控当前真正接管网络的代理路径，不只监控龙猫云。系统代理开启时检查当前代理客户端；使用 TUN 或直连时会区分代理路径故障和本地网络故障。确认断连和恢复后会写入“代理连接记录.csv”。双击“查看代理连接记录.bat”可以打开记录。

四、网络出问题时怎么做

1. 右键托盘盾牌图标；
2. 点击“一键退出全部代理并恢复普通网络”；
3. 查看准备关闭的组件，确认继续；
4. 如果普通权限清理后仍有受保护残留，确认一次 Windows UAC；
5. 等待程序分别显示：
   - 系统代理已关闭：是；
   - 代理残留已清理：是；
   - 普通网络已确认恢复：是；
6. 如需代理，只打开一个客户端；
7. 重启 Codex、终端等仍读取旧代理环境变量的程序。

如果任何一项不是“是”，程序会显示“部分完成”，不会假装已经修好。

五、注意事项

- 不要同时运行多个 FlClash 同源客户端；
- Clash Party 的 `mihomo.exe`会结合安装路径识别；其他路径下无法确认归属的 `mihomo.exe`不会被自动结束；
- 不要依赖修改 7890/7892 来解决同源 Helper 冲突；
- 断网急救不会替你登录、更新订阅、选节点或连接；

六、龙猫云改为 7892 时

1. 7892 可以避开仍占用 7890 的 Clash 核心，减少端口直接冲突；
2. 确认 Windows 系统代理已经指向 127.0.0.1:7892；
3. Codex、终端如果仍保留 127.0.0.1:7890，需要同步代理环境变量并重启；
4. 改端口不能解决系统代理开关、TUN 或 FlClash 同源 Helper 争用；
5. 断连判断跟随当前正在接管网络的客户端或 TUN/直连路径；没有接管网络的客户端不会被当作断连。
- 断网急救不会结束未知进程，不清理未知网卡、未知路由或企业 PAC；
- 节点或线路超时时，只会记录并提示更新节点，不会自动杀核心；
- 当前代理环境变量修改后，已经运行的程序必须重启才能读取新值。

六、如何卸载

双击“卸载断网急救.bat”。

卸载会移除开机启动项并停止托盘监控，但保留日志和备份。卸载高权限 Helper 需要管理员权限。

七、校验下载文件

发布包旁边提供 SHA-256 文件。可以在 PowerShell 中运行：

Get-FileHash .\断网急救_v0.4.2-beta.zip -Algorithm SHA256

将结果与“断网急救_v0.4.2-beta_SHA256.txt”中的值比较，一致后再解压使用。

八、Helper 安装错误

如果看到“创建 Helper 服务失败：1639”，说明使用的是 2026-07-16 修复前的旧压缩包。请删除旧解压目录，换用新的 v0.4.0-beta 压缩包完整解压，再运行“安装高权限Helper_仅需一次UAC.bat”。

如果提示 UAC 被取消，Helper 不会安装；普通监控和当前代理不受影响。重新运行安装文件并在 UAC 窗口点击“是”即可。

如果看到“.NET Framework：另一个程序正在使用此文件”，说明使用的是加入文件锁保护前的旧程序。重新运行新版“安装断网急救.bat”升级即可；新版会重试文件写入，并隔离单次菜单操作异常，不再让托盘监控因此退出。
